## docusearch

Reusable Python package for document search and retrieval utilities.

### Installation

Once this repo is pushed to GitHub, you can install it with:

- **With uv**:

```bash
uv add git+https://github.com/jonmontgomery/docusearch.git
```

- **With pip**:

```bash
pip install git+https://github.com/jonmontgomery/docusearch.git
```

### Usage

After installing:

```python
from docusearch import __version__

print(__version__)
```

You can add your own modules under the `docusearch` package in `src/`.
